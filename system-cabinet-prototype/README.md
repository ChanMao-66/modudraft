# MODUDRAFT 系統櫃獨立原型

此目錄保留系統櫃的獨立開發邊界，已由 MODUDRAFT 首頁進入並共用 Project Schema v1；後續仍可單獨修改與測試，再穩定併入正式流程。

## 高級感 UI V1

- 採用明亮工作室介面、藍灰工具骨架與低飽和木質重點色，維持工程圖清晰度。
- 中央畫布為視覺主角，左右側欄改為分組工具抽屜與屬性檢查器。
- 統一按鈕、輸入框、選取、提示與手機版觸控尺寸。
- 3D 加入地板、背景牆、柔和燈光、陰影、色彩映射與霧面板材。

## 已完成

- 多面牆工作流程：每面牆有自己的寬、高、靠左／置中／靠右排列與牆面物件。
- 立面圖負責整面牆配置；立面配置圖只編輯目前選取的單一櫃體內部。
- 補板已改為獨立牆面物件，可插入、排序、選取、調整與刪除，不再重複加入櫃體寬度。
- 可直接點選櫃體、補板、分格、層板、門片、吊衣桿或抽屜，設定面板會切換至對應內容。
- 每一格可獨立指定「開放格／吊衣桿／抽屜」，並可指定任一格自動補足剩餘高度。
- 3D 有門／無門切換；無門時保留層板、吊衣桿與抽屜。
- 衣櫃推拉門建議成品深度 650 mm；其他用途的推拉門保留自訂深度。
- 多座系統高櫃與用途預設：衣櫃、鞋櫃、書櫃、儲藏櫃。
- 櫃身預設高 2400 mm、腳座 100 mm，頂天花時依天花高度自動計算。
- 對開門：櫃身深 580 mm、門片 20 mm、成品深 600 mm。
- 推拉門：櫃身深 600 mm、前方雙軌 50 mm、成品深 650 mm；左右櫃身側板延伸到 650 mm。
- 中立板：1200 mm 自動分成 800 + 400 mm；1600 mm 自動分成 800 + 800 mm；亦可關閉或自訂格數。
- 層板自動等分，預設上緣間距不超過 300 mm。
- 手動分格以「櫃身高度減一片底板厚度」為總配置高度，尺寸從板上緣量至下一片板上緣。
- 對開門支援外門、內門、整片長門與依分格生門。
- 推拉門固定整片，依總寬自動產生 2～4 片並保留重疊量。
- 吊衣區高度：1000、1500、1650 mm 與自訂。
- 抽屜第一版外觀與基本高度配置。
- 靠牆補邊與見光側封分開計算：補板是牆面獨立物件；側封仍是櫃體全深、落地的外觀板。
- 平面圖、門片立面圖、無門片立面配置圖與 3D。

## 暫定值

- 背板厚度暫定 8 mm，只用於內部結構與 3D；後續可依實際板材改成可調整欄位。
- 推拉門重疊量暫定 30 mm，實際製作時需依選用的軌道五金確認。
- 抽屜目前只處理外觀、數量與單屜高度，尚未加入滑軌與抽盒拆板。
