用手機掃描開啟

使用手機相機掃描後，會從永久 HTTPS 網址直接開啟這套設計軟體。

https://chanmao-66.github.io/modudraft/
下載離線網頁包
  1. Android Chrome：按「立即安裝」，或從瀏覽器選單選擇「安裝應用程式」。
  2. iPhone Safari：按分享按鈕，再選擇「加入主畫面」。
  3. 安裝後可從手機桌面開啟，設計資料仍保留在這台裝置的瀏覽器中。

GitHub Pages 使用 HTTPS，可支援瀏覽器原生 PWA 安裝；離線網頁包是備份檔，不是 Android APK。
